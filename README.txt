== Description ==

Plugin to Generate QR Code with link inserted by the user in front-end with a form:

¿How use the plugin?:

1. Just add the shortcode [qr_link_generator_for_wp_shortcode] to any page where you want the qr code generator to be used.
1. Save the changes.
1. Enjoy!

== Installation ==

1. Download the file and copy it to the path /wp-content/plugins/.
1. Activate the plugin in the list of WordPress plugins.
1. That's all.
